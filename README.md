# De patria a patria / Homeland

## Descripción

Este proyecto es una página web responsiva que muestra una galería de arte y lugares representativos. Su funcionalidad principal es adaptarse fluidamente a múltiples resoluciones de pantalla (Escritorio, Tablet y Móvil) manteniendo una experiencia de usuario óptima.

## Tecnologías y técnicas utilizadas

- HTML5 Semántico
- CSS3 (Flexbox, Posicionamiento relativo/absoluto)
- Metodología BEM (Bloque, Elemento, Modificador)
- Organización de archivos BEM Flat
- Diseño Responsivo (Media Queries: Desktop-first)
